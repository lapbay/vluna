# Security Policy

If you discover a security vulnerability in Vluna OSS, please report it privately.

## Reporting a vulnerability

- Email: `ray@vluna.ai`
- Include:
  - Affected versions/commit
  - Repro steps
  - Impact assessment
  - Potential workaround (if any)

Please do **not** report security issues in public issues or pull requests.

## Response expectations

We will acknowledge valid reports and begin triage within 24–48 hours.

## Disclosure policy

We use coordinated disclosure. Please allow us reasonable time to investigate and patch before public disclosure.
